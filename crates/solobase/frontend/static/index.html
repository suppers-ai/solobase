Built on the WAFER Runtime

Solobase is powered by composable blocks and flows. Each feature — auth, storage, database, products — is a block you can extend, replace, or combine. No vendor lock-in, complete control.

Deploy in seconds

# Download the latest release
curl -sSL https://github.com/suppers-ai/solobase/releases/latest/download/solobase-linux-amd64.tar.gz | tar xz

# Set a JWT secret and run
export JWT_SECRET="your-secret-key-at-least-32-chars"
./solobase

# That's it. Auth, database, storage, products, admin panel — all running on :8090

No Docker, no Node.js, no complex configuration. A single binary with everything built in.

Everything You Need, Built In

Every feature is a block. Enable what you need, disable what you don't. Each block provides APIs, admin UI, and user-facing pages out of the box.

Authentication

Signup, login, JWT tokens, token refresh, API keys, and OAuth (Google, GitHub). Role-based access control with IAM policies.

Database

SQLite out of the box with collections and custom tables. Automatic migrations, full admin query console, and REST API for CRUD.

File Storage

Buckets, file upload, folders, sharing links, and per-user quotas. Local disk or S3-compatible backends with usage tracking.

Products & Payments

Product catalog, groups, purchases, and Stripe checkout with webhooks. Define plans, accept payments, and track subscriptions.

Admin Panel

User management, IAM roles, database browser, request logs, system settings, and block configuration. Full control from one dashboard.

User Dashboard

Ready-made user portal with account settings, plan management, deployments, and API key creation. Works out of the box.
